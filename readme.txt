Курсовая работа по технологиям сетевого взаимодействия.

1) Для запуска необходимо установить модули из requirements.txt
2) Так же необходимо создать контейнер базы данных postgre SQL в Docker или запустить БД на целевой машине
3) Указать в Library\settings.py в разделе DATABASES данные используемой вами БД
4) Провести миграции командой python manage.py makemigrations
5) Запуск api осуществляется путем ввода команды python manage.py runserver (port(не обязательно))
6) После запуска необходимо перейти на http://127.0.0.1:8000(или port);
7) Откроется главная страница со следующим функционалом:
7.1) Поиск по рекомендуемой литературе и шифрам книг с сортировкой результата(если поля поиска пусты то выводится вся таблица)
7.2) Добавить рекомендуемую литературу
7.3) Добавить шифр книги
7.4) Просмотр объединённой таблицы с возможностью сортировки
8) В приложении предусмотрено добавление готовых данных в файле DB_test_info.py для этого выполните следующие команды
8.1) python manage.py shell
8.2) from DB_test_info import bulk_create_RecommendedLiterature
8.3) from DB_test_info import bulk_create_BookCipher
8.4) bulk_create_RecommendedLiterature()
8.5) bulk_create_BookCipher()
8.6) exit()
8.7) Теперь в БД появятся готовые данные
